== badomeow 自定义背景 / Custom Backgrounds ==

在此文件夹放入自定义背景图片。
Place your custom background images here.

支持格式 / Supported: TGA (32-bit uncompressed) 或 BLP
尺寸要求 / Size: 必须是2的幂次 (256x256, 512x256, 512x512, 1024x512)

使用方法 / Usage:
1. 放入 .tga 或 .blp 文件到此文件夹
2. 打开设置 /bm → 自定义背景
3. 输入: Interface\AddOns\badomeow\Textures\Backgrounds\文件名 (不含扩展名)
4. /reload 生效

Example:
  File: druid_art.tga
  Path: Interface\AddOns\badomeow\Textures\Backgrounds\druid_art
